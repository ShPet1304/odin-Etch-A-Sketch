## üßê About The Project
The goal of this project was to create a digital "Etch-a-Sketch" using HTML, CSS, and Vanilla JavaScript. It involves creating a grid of square `divs` that change color when the mouse hovers over them.

### ‚ú® Features
* **Dynamic Grid:** Users can select a canvas size (e.g., 16x16, 64x64) via a prompt.
* **Drawing:** Hovering over the grid cells leaves a pixel trail.
* **Reset:** A button to clear the screen and resize the grid.
* **Rainbow Mode:** Randomizes the color of each pixel.
